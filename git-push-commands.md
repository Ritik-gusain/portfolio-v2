# Git Commands Used to Push the Project

```bash
# 1. Initialize a new git repository
git init

# 2. Stage all files
git add .

# 3. Commit with a message
git commit -m "Initial commit: portfolio project"

# 4. Add the remote GitHub repository
git remote add origin https://github.com/Ritik-gusain/portfolio-v2.git

# 5. Rename branch to main
git branch -M main

# 6. Push to GitHub and set upstream tracking
git push -u origin main
```
